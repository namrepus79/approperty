Approperty

連接 Supabase 雲端資料庫

將資料儲存在雲端，讓整個團隊可以從任何裝置存取。設定約需 15 分鐘。

Supabase 免費方案支援最多 500MB 資料及無限團隊成員，無需信用卡。
略過，使用示範模式

建立 Supabase 專案

首先在 Supabase 建立免費帳號與專案。

  1. 前往 supabase.com 免費註冊
  2. 點擊 New Project（新專案）
  3. 輸入名稱（例：Approperty）並設定強密碼
  4. 選擇離台灣最近的區域（建議 Tokyo 或 Singapore）
  5. 點擊 Create new project，等待約 2 分鐘完成建立
略過

建立資料庫資料表

在 Supabase 專案中開啟 SQL Editor，貼上以下指令並執行。

-- Approperty 資料庫結構 create table properties ( id bigint generated always as identity primary key, name text not null, address text, type text, size numeric, rent int, tenant text, status text default '空置', lease_start date, lease_end date, payment text default '待繳中', notes text, created_at timestamptz default now() ); create table groups ( id text primary key, name text not null, type text, description text, icon text, color text, property_ids int[] default '{}' ); create table maintenance ( id bigint generated always as identity primary key, property_id bigint references properties(id) on delete cascade, title text not null, description text, priority text default '中', status text default '待處理', cost int default 0, reported_date date, created_at timestamptz default now() ); create table expenses ( id bigint generated always as identity primary key, property_id bigint references properties(id) on delete cascade, category text, description text, amount int not null, expense_date date, created_at timestamptz default now() ); create table contacts ( id bigint generated always as identity primary key, name text not null, role text, email text, phone text, company text, property_id bigint references properties(id) on delete set null, notes text, created_at timestamptz default now() ); alter table properties enable row level security; alter table groups enable row level security; alter table maintenance enable row level security; alter table expenses enable row level security; alter table contacts enable row level security; create policy "Allow all" on properties for all using (true) with check (true); create policy "Allow all" on groups for all using (true) with check (true); create policy "Allow all" on maintenance for all using (true) with check (true); create policy "Allow all" on expenses for all using (true) with check (true); create policy "Allow all" on contacts for all using (true) with check (true);
在 Supabase 中：前往 SQL Editor → New query，貼上指令後點擊 Run（執行）。
